DNS

DNS (Domain Name System) translates human-readable domain names into IP addresses. Our domain resolves to multiple IPv4 and IPv6 addresses, demonstrating redundancy and load balancing.

IP Addressing

The website resolves to both IPv4 and IPv6 addresses. IPv4 uses 32-bit addressing while IPv6 uses 128-bit addressing for larger address space.

HTTP vs HTTPS

This website uses HTTPS, which encrypts communication using TLS over TCP/IP. HTTP traffic automatically redirects to HTTPS, ensuring secure communication.
